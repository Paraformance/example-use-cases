Mandelbrot_set
==============

Here you could find the code for **The Mandelbrot set in C++11**, for more informations visit the project webpage:

http://solarianprogrammer.com/2013/02/28/mandelbroot-set-cpp-11/

### Requirements
1. Visual C++
2. FreeImage library
3. Intel TBB
4. OpenMP

### Visual Studio
Open mandlebrot.sln
